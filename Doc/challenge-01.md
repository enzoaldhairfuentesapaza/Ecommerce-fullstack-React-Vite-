# Challenge 01 – Add to Cart Animation

## 1. Objetivo
Agregar un **feedback visual claro e inmediato** al botón “Add to Cart” cuando el usuario añade un producto, siguiendo exactamente el comportamiento definido en el mockup `01-add-to-cart-animation.html`.

## 2. Qué se construyó
Se implementó una animación con cambio de escala, color y checkmark que se reinicia automáticamente.

## 3. Arquitectura y decisiones técnicas
Se creó `ProductCard` y `AddToCartButton` para aislar lógica y UI.

## 4. Flujo de funcionamiento
El botón entra en estado `.adding`, anima y vuelve al estado normal.

## 6. Requisitos cumplidos
Scale, color, check, reset automático, clicks rápidos.
