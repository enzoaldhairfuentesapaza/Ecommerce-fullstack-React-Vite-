# Challenge 02B – Order Detail View

## 1. Objetivo
Implementar una vista dedicada de **detalle de orden** accesible mediante la ruta `/orders/:orderId`, permitiendo al usuario visualizar toda la información de una orden específica luego de haberla creado o desde el historial.

## 2. Qué se construyó
Vista completa de orden con productos, totales y estado visual.

## 3. Arquitectura y decisiones técnicas
Uso de react-router-dom, separación por páginas y matching de productos para imágenes.

## 4. Flujo de funcionamiento
Crear orden → Modal → View Order → Vista de detalle.

/test-videos/challenge-02b.mp4

## 6. Requisitos cumplidos
Routing, UI, badges, responsive.

## 7. Resultado
Vista profesional y escalable.
